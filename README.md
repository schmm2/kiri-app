## Development

Local installation

- Git
- Yarn
- Nodejs

### Setup

- git clone
- yarn install
- Create .env file
  - Add backend URI: REACT_APP_BACKENDAPIURL="localhost:7071/api"
  - Add auth app id: REACT_APP_AUTHAPPID="xxxx"
